## Known Limitations

1. **File Format** - Only Flowgorithm .fgl files supported
2. **Analysis** - Pattern-based detection, not deep learning
3. **Scalability** - Single database instance
4. **Security** - No authentication implemented yet
5. **Real-time** - No WebSocket support for live updates
