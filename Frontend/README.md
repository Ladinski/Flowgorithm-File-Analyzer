# Flowgorithm Analyzer Pro - Frontend

Beautiful React frontend for the Flowgorithm file analysis system.

## Features

- 🎨 Modern UI with Tailwind CSS
- 📊 Real-time dashboards and charts
- 📤 File upload (single and bulk)
- 📈 Student performance tracking
- 🔍 Detailed analysis visualization

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Server runs at `http://localhost:3000`

## Build

```bash
npm run build
```

## Environment Variables

Create `.env` file:

```
REACT_APP_API_URL=http://localhost:5000
```

## Technologies

- React 18
- TypeScript
- Tailwind CSS
- Recharts for visualization
- Axios for API calls
- React Router for navigation
